# Сервис графического представления задачи 3N + 1
## Описание проекта
- Framework - Sinatra
- Web Server - Puma

## Endpoints
- /max - Максимальные 20 последовательностей для 1_000_000 чисел
- /all - Последовательности 30_000 чисел представленные на 3 графиках

## Запуск проекта

```bash
# Запустить на localhost:3000
docker-compose up
```
